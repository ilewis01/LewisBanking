Introduction:

Banking System is developed in Python to replaced manual System by computerized system.

This system allows to create an new account and Allows to deposit and withdrawal amount facilities.

It Also maintain Loan Transaction and Paying Loan Installment.

It also Provide Advanced searching Facilities.

Modules:
  1. Open Account
    User to Create The New Account Of Customer
  2. Deposit Amount
    In This Module Customers Can Deposite Amount in their Accounts.
  3. Withdraw Amount
    In This Module Customers Can Withdrawal Amount from their Accounts.
  4. Display All Accounts
    This Module Display All The Account Information.
  5. Display Transaction
    This Module Display Transaction Information By Account No, By Date And All Transactions of All Accounts.
  6. Delete Account
    This Module Closed Account Of Customers.
  7. Loan
    This Module Maintain All Transaction Of Loan. Also Maintain Installment.
  8. Search
    This Module Provide Advanced Searching Techniques.
  9. Transfer Amount
    This Module Provide Facility For Transfering Amount From One Account To Another Account.
Validations:
  • System Generate Account No and Loan Id Automatically.
  • User can not Enter number In Name Field.
  • User can not Enter Character In Mobile No.
  • User can not Enter Less or More Than 10 digits in Mobile no.
  • User can not Enter Character In Numeric Field.
  • User can not Enter Invalid choice and Invalid date into the system.

This application was developed as a Python class project by Immanuel Lewis who is the sole author and ower.
Key functionaliy includes:

  • User creation and data storage
  • Both checking and savings account creation and generation of random account numbers consisting of integers only
  • Loan creation. This feature randomly generates a loan decision, ramdomly generates a loan ID consisting of 8 alpha-numeric characters, a random interest rate between 0.01 abd 0.05
  • A general search algorithm that will return items matching a single search criteria if any exist
  • An advanced search algorithm that will return either items within a range of dates or a range of transaction amounts
  • Deposit funds into checking or savings account
  • Withdraw funds from checking or savings account
  • Transfer funds between accounts
  • View transaction history of loans or accounts
  • Make payments on outstanding loan balances
  • Refinance loans (The new rate is generated randomly
  • View scheduled payment dates for loans
  • Sort results by several pertinent class members including date, balance, etc.
  • Store and maintain all account and loan history
  • Recover forgotten passwords
  • Change profile information
  • Users also have the ability to delete savings and checking accounts
  • User must open an account to create a profile
